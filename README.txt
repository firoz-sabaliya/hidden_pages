Description
-----------
HIdden Pages module is used for to access certain pages using API key. API key can be set from site information page. 
Hidden Pages can be accessed from http://example.com/page_json/<SiteAPiKey>/<NodeID>

Configuration
-------------
API key can be set from here
Configuration > Site Information 
or at admin/config/system/site-information

Limitation
----------
First argument must be page_json
Second argument must be API key
Third argument should be existing Node id (Numeric value only)
Content Type of Node should be Page type
